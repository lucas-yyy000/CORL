# CORL (Clean Offline Reinforcement Learning)

## Getting started

```bash
git clone https://github.com/tinkoff-ai/CORL.git && cd CORL
pip install -r requirements/requirements_dev.txt
```
(downgrade cython if necessary with ```pip install "cython<3"```)

## Citing CORL
